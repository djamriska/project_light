This project would not have been possible without a good number of external sources we located on the web. Here arhe the most relevant

BM25 Source Code
The base of our application is a copy of the BM25 algorithm we located on the internet. We located a large number of github sources, but this was the best version we found. It didn't run from the origianl download, but it was a great starting point.
https://github.com/nhirakawa/BM25

D3 (Data-Driven Documents)
A JavaScript library for producing dynamic, interactive data visualizations in web browsers. We used this to display the date results as it can be run locally using a simple HTTP server. A link to the source and examples can be found here,
href=https://d3js.org.
