FAQ

  • Q: Why are there multiple files listed for each date/time
  • A: The query file supports the ability to submit multiple queries, the results show a Q# for each query that it found. In some cases that may be an empty query, where the terms were not listed on the that line

  • Q: Some of the articles seem like they should be higher in the ranking, what gives?
  • A: This application uses a custom weight for each term to determine how important it is. These settings are controled by the user in the ../text/weights.txt file. In a future version we will provide the ability for the end user to personal weight rankings.

  • Q: What does terms and docs mean?
  • A: Terms is the number of terms submitted in the query. Docs is the number of documents found that match one or more terms in the query.
